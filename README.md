# sysadmin-scripts
A collection of Bash scripts for Linux system administration tasks including user management, service monitoring, backups, and more. Built as part of my LFCS prep and ongoing learning in Linux and automation.

This repo is organized by task area. Each directory includes a dedicated `README.md` for usage details.

- **Backup Scripts** – Simple backup routines using `rsync`, `tar`, etc.
- **System Monitoring** – Check system services, restart them if needed, and send alerts
- **Networking Tools** – Update IP addresses, restart interfaces, and log changes
- **Package Management** - Checking for updates, installing updates, and removing unneeded packages
- **Study Logs** - Documented projects, homelabs, etc 
- **User Management** – Create, delete, and manage users and groups

---

## System Compatibility

- Scripts are written in **Bash**, tested primarily on **RHEL-based systems** (Rocky Linux, AlmaLinux, CentOS)
- Most should work on **any Linux distribution** with minor adjustments if needed
