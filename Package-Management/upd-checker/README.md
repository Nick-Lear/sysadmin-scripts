# upd-checker.sh README
***************
## What does this do?
This script automates the process of checking for available package updates, upgrading available packages, and cleaning up after itself when it's completed.
 
## How to / Usage
Superuser permissions are required, otherwise run the command as is, there are no additional options or arguments available.

## Setting permissions

In order for the command to be executable, it will need to be given execute permissions via:

`sudo chmod +x upd-checker.sh`
OR
`sudo chmod 755 upd-checker.sh`


## Syntax

Example with minimum required options:

`./upd-checker.sh`
